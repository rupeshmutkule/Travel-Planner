========================================
API ROUTES
========================================

Backend Server (http://localhost:3001 or https://travel-plane-maker.vercel.app)

Authentication:
  POST   /api/auth/register          # Register new user
  POST   /api/auth/login             # Login user
  POST   /api/auth/send-otp          # Send OTP email

Travel Planning:
  POST   /api/plan                   # Generate travel plan
  GET    /api/history                # Get user's travel history
  POST   /api/history/save           # Save travel plan
  PATCH  /api/history/:id            # Update history item
  DELETE /api/history/:id            # Delete history item

Health Check:
  GET    /api/health                 # Server health status
